Water Fountain Music Visualizer

Embedded Systems Capstone Project (C++, PCB design, Microcontroller)

Back to Projects

About

For this capstone project, my project partner (Beverly Yee) and I decided to make a music visualizer using water pumps to represent the music frequencies.

This project consists of a specially designed PCB, original software to analyze the audio input in real time, a Microcontroller (Teensy 4.0) to run the code and read audio through an aux cord, and a 3d printed enclosure with an acrylic tank for the water.
