About

Also known as a Toon shader, this software is a non-photorealistic shading that makes the object look like something out of a cartoon. It renders a 3d object (any .obj file) fewer colors to render the levels of shading as well as creating sharper changes in color.



Silhouette shading: Creates a larger black silhouette of the object that is rendered behind the main object in order to create a black border like how a comic object/character is draw


Contours: provides more emphasis to the change of color by detecting and adding a black line along the edge of the color change.
