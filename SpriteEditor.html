Sprite Editor

Software application (C++, QTcreator)

Back to Projects

About

This project implements a primary window for editing each frame of the sprite as well as a seperate window to preview the sprite. Each frame object is stored as a vector array and displayed using a pixel grid. The user can save the current sprite to a json file or load a json file to open an existing sprite. The preview window animates the full sprite at a user defined framerate.
